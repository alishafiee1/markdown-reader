# آرشیو — امتیاز و نظر کاربران (user-reviews)

## چی بود؟

ستاره و نظر برای هر رستوران — علی قبل از سفارش ببیند دیگران چی گفته‌اند.

---

## چرا ولش کردیم؟

- **Moderation:** نظرات جعلی و توهین — بدون تیم نگهبان اعتماد از بین می‌رود.
- **تمرکز MVP:** اول منوی به‌روز و تخفیف؛ امتیاز رقیق می‌کند.
- رستوران‌های کوچک ممکن است از نظر منفی حساس باشند — فعلاً نمی‌خواهیم وارد شویم.
- در پروپزال اصلی §۵ «امتیاز کاربران» عمداً خارج بود.

---

## چی جایگزین کردیم؟

- توضیح کوتاه از خود رستوران روی کارت
- تخفیف امروز به‌عنوان سیگنال «ارزش دیدن»
